### 6.1 运动学公式的推导
(1) 描述运动的物理量
		  ① 描述匀速运动一共有 3 个物理量，分别是：位移、时间、速度.
		  ② 描述匀变速直线运动一共有 5 个，分别是：v0、vt、a、x、t.
(2) 公式推导
| 条件     | 公式                     |
| --------- | ---------------------- |
| 缺少 x   | vt = v0 + at          |
| 缺少 a   | x = [(vo+vt) t]/2    |
| 缺少 v0 | x = vtT - 1/2aT^2  |
| 缺少 vt  | x = v0T + 1/2aT^2 |
| 缺少 t   | 2ax = vt^2 - v0^2   |
